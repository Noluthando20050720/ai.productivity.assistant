# AI Workplace Productivity Assistant

## Project Overview

The **AI Workplace Productivity Assistant** is a modern, responsive web application designed to help professionals automate everyday workplace tasks using Artificial Intelligence. The application provides AI-powered tools to generate professional emails, create organised task schedules, and assist users through an interactive workplace chatbot.

The application is designed with simplicity in mind. Users can access all features immediately without creating an account or logging in. No user data is stored, making it lightweight and easy to use.

---

## Features Implemented

### 📧 Smart Email Generator
- Generate professional emails using AI
- Support for multiple tones:
  - Formal
  - Friendly
  - Persuasive
- Editable AI-generated content
- Copy generated emails
- Regenerate responses

### 📅 AI Task Planner / Scheduler
- Generate daily or weekly schedules
- Prioritise tasks based on importance
- Create realistic work plans with breaks
- Editable schedules
- Regenerate schedules

### 🤖 AI Workplace Chat Assistant
- Interactive AI chatbot for workplace productivity
- Answer workplace-related questions
- Assist with planning and brainstorming
- Help users write professional content
- Provide suggested prompts for common tasks

### 🎨 Modern Dashboard
- Responsive SaaS-inspired interface
- Sidebar navigation
- Quick action cards
- Mobile-friendly design

### ⚠ Responsible AI
- AI disclaimer encouraging users to review generated content before use

## Setup Instructions

### Prerequisites

Before running the project, ensure you have:

- Node.js (v18 or later)
- npm installed
- Git installed

---

### Clone the Repository

```bash
git clone https://github.com/yourusername/ai-workplace-productivity-assistant.git
```

---

### Navigate to the Project Folder

```bash
cd ai-workplace-productivity-assistant
```

---

### Install Dependencies

```bash
npm install
```

---

### Configure Environment Variables

Create a `.env` file in the project root and add your OpenAI API key:

```env
VITE_OPENAI_API_KEY=your_api_key_here
```

---

### Start the Development Server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
